# Desejo Natural – Mini ERP

`CURSO: Sistemas de Informação`

`DISCIPLINA: Projeto - Trabalho de Conclusão de Curso`

`SEMESTRE: 8º`

`INSTITUIÇÃO: Pontifícia Universidade Católica de Minas Gerais`

Transformação digital em microempresas: desenvolvimento de um mini ERP para as geleias Desejo Natural.

Este projeto representa o artefato de software entregue ao cliente como um aplicativo desktop através do empacotamento com Electron. O uso deste framework nos possibilitou desenvolver um ERP simplificado e sem a introdução de custos adicionais à folha do cliente, entregando um front React que servido pela API em Java Quarkus, consome um banco SQLite para formar uma solução simples e robusta que atende às necessidades de um pequeno negócio.
